School Result Portal

Supports WAEC or NECO Result PIN Checker

Overview

Built with Laravel 8.54, supporting PHP 7.4+. The portal includes a clean admin dashboard and supports bulk upload of data into the database. The portal is ideal for school result management, allowing students to access their termly results using generated PINs. Customization options are available to remove the PIN generation feature upon payment.

How to Use

  1. Log in to the admin dashboard using your credentials.
  2. Upload bulk data using the provided templates.
  3. Generate PINs (if applicable) for student result access.
  4. Share the generated PINs with students or guardians for result checking.
  5. Customize the portal settings as per your school's requirements.
